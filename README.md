🛡️ Hoaxbomb Threat Detection
📌 Overview


Hoaxbomb is a cybersecurity-focused prototype developed during a hackathon to detect potentially misleading or malicious content using rule-based analysis. 
The system focuses on identifying suspicious text patterns efficiently under constrained development time.

🚀 Features

Detects hoax or suspicious text using keyword and pattern matching
Performs input validation for safer data handling
Lightweight and efficient rule-based detection logic
Built as a functional prototype under hackathon constraints

🧠 How It Works

User provides input text
System processes input using predefined rules
Pattern matching and keyword analysis are applied
Suspicious or hoax content is flagged as output

⚙️ Tech Stack

Python
Rule-based pattern matching
